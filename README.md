# Penny - personal finance chatbot
a personalized expense analysis and budget planning chatbot powered by Google gemini

## Techstack used 
1. Tailwind CSS
2. Vite + ReactJS
3. Google Gemini API

## setup 
1. clone the project from github by running `git clone <project_url>`
2. Go inside the project directory , use `cd <project_folder_name>` 
3. install the required dependencies by running `npm install`
4. create a `.env` file at the root of project , refer the `.env.example` file
5. Put your gemini API key in the `.env` file
6. From the root of project , run `npm run dev` . Your project should be up and running on port 5173 

Project should be up and running on - ```http://localhost:5173/```
